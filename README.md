# Template Method - Basquete

Implementação do padrão de projeto **Template Method** aplicado ao contexto de testes de jogadores de basquete.

## O que é?

Sistema que avalia jogadores de basquete através de testes, com critérios diferentes de aprovação:
- **Jogador Juvenil**: aprovado com média ≥ 6.0
- **Jogador Profissional**: aprovado com média ≥ 7.0

## Como executar?

```bash
cd basquete
mvn test
```
